## The Daily MCP Cycle

Perform this 6-step cycle daily to maintain system hygiene and alignment.

1.  **Repo Audit**: Check for uncommitted changes and branch drift.
2.  **Supabase Health**: Verify database connectivity and migration status.
3.  **Open PR Checks**: Review active Pull Requests for conflict or staleness.
4.  **Issue Correlations**: Link new issues to existing code paths or VaultNodes.
5.  **MCP Sanity Tests**: Run the Basic Connectivity test from the Toolkit.
6.  **Patch Staging**: Generate and review patches for any low-hanging fruit.

> "Give me the daily MCP cycle."
